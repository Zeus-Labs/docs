ZeusCloud provides 100+ predefined security rules for AWS environments, and it also maps them to various compliance frameworks. Currently, these rules span cloud misconfigurations and common, identity-based attack paths.

The security rules span the following AWS services
- CloudTrail
- CloudWatch
- EC2
- ElasticSearch
- ELBv2
- Lambda
- IAM
- KMS
- RDS
- S3
- Secrets Manager
- Security Hub
- SQS
- VPC
