title: "CloudWatch"
---

ZeusCloud provides the following security rules for AWS CloudWatch:
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/aws_console_auth_failures.go">Log metric filter and alarm should exist for AWS Mangement Console authentication failures</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/bucket_policy_changes.go">Log metric filter and alarm should exist for S3 bucket policy changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/cloudtrail_configuration_changes.go">Log metric filter and alarm should exist for Cloudtrail configuration changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/cmk_disable_or_delete.go">Log metric filter and alarm should exist for disabling or scheduling deletion of KMS CMKs</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/config_configuration_changes.go">Log metric filter and alarm should exist for AWS Config configuration changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/iam_policy_changes.go">Log metric filter and alarm should exist for IAM policy changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/nacl_changes.go">Log metric filter and alarm should exist for Network Access Control List (NACL) changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/network_gateway_changes.go">Log metric filter and alarm should exist for network gateway changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/organization_changes.go">Log metric filter and alarm should exist for AWS Organization changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/root_account_usage.go">Log metric filter and alarm should exist for usage of the root account</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/route_table_changes.go">Log metric filter and alarm should exist for route table changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/security_group_changes.go">Log metric filter and alarm should exist for security group changes</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/sign_in_without_mfa.go">Log metric filter and alarm should exist for management console sign in without MFA</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/unauthorized_api.go">Log metric filter and alarm should exist for unauthorized API calls</a>
- <a href="https://github.com/Zeus-Labs/ZeusCloud/blob/main/backend/rules/cloudwatch/vpc_changes.go">Log metric filter and alarm should exist for VPC changes</a>
